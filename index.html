About MECHA // BLOCKS


MECHA // BLOCKS [ MechaBlocks ]
is a collection of custom LEGO® mecha MOCs
original model designs modeled using open source technology
and featuring interactive 3D model viewers, building instructions,
all inspired by anime, science fiction, and video games.
